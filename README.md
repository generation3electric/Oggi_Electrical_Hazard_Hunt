# Oggi’s Electrical Hazard Hunt

A mobile-first hidden-object game for GEN3 Electric & HVAC. Players explore five
illustrated Philadelphia homes with Oggi, find electrical hazards, earn paw
ratings, and share their results.

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

The project is a standard Next.js static-export app and can be deployed directly
to Vercel.
